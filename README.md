# Proyecto de Procesamiento de Lenguaje Natural (NLP) - Unidad 1

Este repositorio contiene el desarrollo de la primera actividad práctica, donde se personalizan notebooks de procesamiento de texto y análisis de sentimientos.

## 🛠️ Especificaciones Técnicas
* **Lenguaje:** Python 3.12
* **Librerías principales:** spaCy (`es_core_news_sm`), NLTK (VADER), Pandas y Matplotlib.

## 📂 Archivos del Repositorio
* `Jhon_NPL_basics.ipynb`: Análisis gramatical (POS Tagging) aplicado a un texto técnico de salud.
* `Jhon_Sentiment_Libros.ipynb`: Clasificación de sentimientos sobre reseñas de libros.
* `resumen_fisioterapia.txt`: Corpus externo sobre crioterapia y rehabilitación.
* `reseñas_libros.txt`: Dataset personalizado de críticas literarias.

## 📈 Hallazgos Destacados
* **NLP Basics:** Se identificó que en textos de fisioterapia predominan los sustantivos técnicos, lo que facilita la extracción de términos clínicos.
* **Sentimientos:** Se observó que el modelo VADER tiene limitaciones con el vocabulario neutro en español, lo que se analizó mediante una Matriz de Confusión.
